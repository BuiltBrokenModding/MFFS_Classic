# Scale Module

The Scale Module can be placed in the upgrade slots of various MFFS machines to upgrade their distance capabilities.

Its effects vary depending on the machine, but most commonly include:
- Incrasing force field size
- Increasing interdiction area
- Increasing the fuel boost in a [](@mffs:coercion_deriver)
