# Anti-Personnel Module

The Anti-Personnel Module is an optional module for the Interdiction Matrix.

<PrefabObtaining />

## Effect

With this module installed, the Matrix will automatically detect unauthorised players that enter its action range and
kill them - almost instantly.

Friendly and hostile mobs are not harmed.
