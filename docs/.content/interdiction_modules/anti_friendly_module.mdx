# Anti-Friendly Module

The Anti-Friendly Module is an optional module for the Interdiction Matrix.

<PrefabObtaining />

## Effect

With this module installed, the Matrix will automatically detect friendly entities (like sheep, cows and villagers) that
enter its scanning range and kill them - almost instantly. Players and hostile mobs are unharmed.
