# Warn Module

The Warn Module is an optional module for the [](@mffs:interdiction_matrix).

<PrefabObtaining />

## Effect

The Warn Module can be placed in the matrix slots of an Interdiction Matrix to periodically send a warning to all
unathorized players within its Action Range.
