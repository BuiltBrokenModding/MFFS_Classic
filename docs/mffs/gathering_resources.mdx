The first step towards force fields is finding the base resources.

MFFS components are mainly made out of Steel Ingots, which can be crafted in 2 simple steps.
First, make yourself some Steel Compound in crafting, which you can then smelt into ingots.

One of the most common ingredients you'll be using in MFFS. It is used to craft base components such as the
Focus Matrix or the Blank Card, as well as force field machinery.
